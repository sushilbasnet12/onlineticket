# ONLINE TICKET BOOKING SYSTEM

#### How to run

* Create a new virtual environment

```console
python -m venv venv
```

* Activate Virtual Environment
  
```
venv\Scripts\activate.bat (cmd)
```

* Install dependencies
  ```
   pip install -r requirements.txt 
  ```

* Run Migrations

* Run Server


--- 
While new push run

pip freeze > requirements.txt
